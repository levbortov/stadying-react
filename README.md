- [React](https://reactjs.org/)

### Библиотеки

- [React Router](https://reactrouter.com/)

### Инструменты

- [Vite](https://vitejs.dev/)
- [Eslint](https://eslint.org/)
- [Prettier](https://prettier.io/)
